Business App Template (React Native)

A clean and modern mobile app template designed for company and local business use cases. This template is ideal for showcasing services, company information, and contact details.

--------Features--------
Clean and modern UI
Home screen with company overview
Services listing screen
Contact screen (email, phone, location UI)
Reusable components
Simple navigation structure
Optimized for scalability
---------Tech Stack----------
React Native (Expo)
JavaScript (ES6+)
React Navigation
Functional Components & Hooks
